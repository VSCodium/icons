Contributing
============

Dependencies
------------

- icns2png
- imagemagick
- png2icns
- icotool
- librsvg
- bc
- pastel


Color Template
--------------

### `normal.conf`

```bash
COLOR_PRIMARY="#62A0EA"
COLOR_MEDIAN1="#3584E4"
COLOR_MEDIAN2="#1C71D8"
COLOR_FLOOR="#1A5FB4"
```

Currently, the color `COLOR_MEDIAN1` is the color `COLOR_PRIMARY` but darkened 2 times (`pastel darken 0.1 $COLOR_PRIMARY`)
